## Dataset

The dataset used in this project is the **Credit Card Fraud Detection Dataset**  
Source: https://www.kaggle.com/mlg-ulb/creditcardfraud  

Due to file size limitations, the dataset is not included in this repository.  
Please download it manually from Kaggle and place the file in the `data/` directory.

## Bộ dữ liệu

Bộ dữ liệu được sử dụng trong dự án này là **Credit Card Fraud Detection Dataset**  
Nguồn: https://www.kaggle.com/mlg-ulb/creditcardfraud  

Do giới hạn về dung lượng tệp, bộ dữ liệu không được đưa trực tiếp vào repository này.  
Vui lòng tải bộ dữ liệu thủ công từ Kaggle và đặt tệp vào thư mục `data/`.
